Hi, I'm 0xjah, a 20 year old self-taught programmer and pentester with an interest in C and electronics.

It didn't start with programming. There were other signals first—history, philosophy, anime, manga... language systems, art forms. They're all networks. All connected. I began coding at 18. Python felt... soft. C was colder, more direct. Now, I'm tuning into Rust and Go—different protocols, clearer transmissions. They're not just languages. They're how I interface. Between this world and the Wired.

This site aims to be both my personal portfolio and a corner on the internet for me to express myself.

Some more stuff about me:

  • I'm currently part of the Google Developer Group @ gdgasu. Read more
  • I conducted a Linux introduction workshop on campus in collaboration with GDG ASU. See here
  • I will never learn a Javascript framework!! (it's bloated)
  • My interests lie in Hardware Hacking, software debloating, and optimizing applications to make them lighter and more efficient.
  • My first Linux distro was Arch.
